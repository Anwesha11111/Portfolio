# Professional Portfolio Website

A modern, multi-page portfolio website showcasing projects, achievements, designs, artwork, and more.
Visit my website:https://portfolio-two-nu-3zjvmjxpiy.vercel.app/#home

## Features

- 10 Different pages (Home, About, Projects, AI Projects, Designs, Artwork, Video Edits, Achievements, Hobbies, Contact)
- Smooth animations and transitions
- Professional dark theme design
- Fully responsive
- Easy to customize

## How to Run

1. Clone this repository
2. Open `index.html` in your browser
3. Or use a live server for better experience

## Customization

Edit the following in `index.html`:
- Your name and tagline
- About me content
- Project details
- Contact information
- Skills percentages

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

This site can be easily deployed to GitHub Pages for free hosting.
